OnAllLayers
ez-clang is an experimental C++ REPL for bare-metal embedded devices. It cross-compiles code on the fly with Clang and uses a LLVM remote-JIT backend to link it incrementally on the device. All heavy lifting happens on the host machine and thus the RPC endpoint on the device is very simple. It only takes few kilobytes of flash memory. Right now, ez-clang supports a selection of ARM Cortex-M devices like Arduino Due and Stellaris LM3S811 (QEMU). Read more >>
